# 🔐 QVault: Quantum Secure Cloud Storage Simulator

## 📌 Project Overview

QVault is a secure cloud storage system that combines **Quantum Key Distribution (QKD)** and **AES encryption** to provide highly secure data storage and sharing. It simulates quantum-safe communication using the BB84 protocol and ensures confidentiality, integrity, and authentication of files stored in the cloud.

---

## 🚀 Key Features

* 🔑 **Quantum Key Distribution (BB84 Protocol)**
* 🔒 **AES Encryption for secure file storage**
* 📧 **Email alerts for security notifications**
* 🔐 **OTP-based download verification**
* 📊 **Audit logs and attack detection**
* ☁️ **Simulated multi-cloud storage (Cloud A & Cloud B)**
* 👤 **User authentication system**

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Encryption:** AES Algorithm
* **Quantum Simulation:** BB84 Protocol
* **Database:** SQLite
* **Email Service:** SMTP

---

## 🏗️ System Architecture

The system consists of:

* User Interface (Login, Upload, Download)
* Authentication Module
* QKD Module (BB84 Protocol Simulation)
* AES Encryption Module
* Cloud Storage (Simulated)
* Email Notification System
* Audit & Monitoring System

---

## 🔄 Workflow

1. User registers and logs in
2. File is uploaded
3. QKD generates secure key
4. File is encrypted using AES
5. Encrypted file stored in cloud
6. On download, OTP verification is required
7. File is decrypted and provided to user
